Privacy Policy

Last updated: March 25, 2026

Overview

SalaryLens is a browser extension that shows salary data from the Bureau of Labor Statistics (BLS) on job listing websites. We are committed to protecting your privacy.

Data We Collect

We collect no personal data. Specifically:

  • No user accounts or registration required
  • No browsing history tracked
  • No analytics or telemetry
  • No cookies (except those set by the payment processor for Pro purchases)
  • No data sent to our servers

Data Stored Locally

SalaryLens stores the following data locally on your device using Chrome's built-in storage API:

  • Saved job offers (job title, company, location, salary, market data)
  • Extension settings (optional API keys for BLS/CareerOneStop)
  • Daily lookup counter (for free tier rate limiting)

This data never leaves your browser. You can delete it at any time by removing the extension.

Third-Party Services

  • Bureau of Labor Statistics API — We send job occupation codes and area codes to the BLS public API to retrieve salary data. No personally identifiable information is included in these requests.
  • CareerOneStop API (optional) — If configured, occupation data requests are sent to the CareerOneStop API. No personal data is transmitted.
  • ExtensionPay / Stripe — If you purchase SalaryLens Pro, payment is processed by Stripe through ExtensionPay. We do not see or store your credit card information. See Stripe's Privacy Policy.

Permissions

SalaryLens requests only the minimum permissions needed:

  • activeTab — Read the current job listing page to extract job title and location
  • storage — Save your offers and settings locally
  • Host permissions for API endpoints — Connect to BLS and CareerOneStop APIs only

Children's Privacy

SalaryLens is not directed at children under 13. We do not knowingly collect information from children.

Changes to This Policy

We may update this policy from time to time. Changes will be posted on this page with an updated date.
